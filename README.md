## To load a page on github pages  
the following commands are used:

```
$ cd dist
$ echo  " dist/ "  >> .gitignore
$ git add --all
$ git commit -m " Deploy to gh-pages "
$ git push origin gh-pages
$ git worktree add dist gh-pages
$ rm -rf dist
```
